# üîê **Authentication**

Secure your AI-powered security scanning with robust authentication. Enforster AI supports multiple authentication methods for different use cases.

## üéØ **Authentication Methods**

<Tabs>
  <Tab title="üîë API Key ">
    **Simple and secure for server-to-server communication**

    ```
    curl -X GET https://api.enforster.ai/v1/scans \
      -H "Authorization: Bearer YOUR_API_KEY"
    ```

    <Steps>
      <Step title="Generate API Key">
        Go to [Dashboard ‚Üí Settings ‚Üí API Keys](https://app.enforster.ai/settings/api)
      </Step>
      <Step title="Copy Your Key">
        Copy the generated key (starts with `ak_live_`)
      </Step>
      <Step title="Add to Headers">
        Include in `Authorization: Bearer YOUR_API_KEY` header
      </Step>
    </Steps>
  </Tab>
</Tabs>

## üõ°Ô∏è **Security Best Practices**

<Warning>
  **Never expose API keys in client-side code, public repositories, or logs\!**
</Warning>

<AccordionGroup>
  <Accordion title="Key Management" icon="key">
    - **Store securely** in environment variables
    - **Rotate regularly** (every 90 days recommended)
    - **Use different keys** for different environments
    - **Revoke immediately** if compromised
  </Accordion>
  <Accordion title="Network Security" icon="globe">
    - **HTTPS only** - never use HTTP
    - **IP whitelisting** for production APIs
    - **VPN access** for sensitive operations
    - **TLS 1.2\+** minimum encryption
  </Accordion>
  <Accordion title="Monitoring" icon="chart-line">
    - **Log API usage** patterns
    - **Monitor failed attempts**
    - **Set up alerts** for unusual activity
    - **Regular security audits**
  </Accordion>
</AccordionGroup>

## üîß **Implementation Examples**

<AccordionGroup>
  <Accordion title="Python" icon="python">
    ```python
    import requests
    import os
    
    API_KEY = os.getenv('ENFORSTER_API_KEY')
    BASE_URL = 'https://api.enforster.ai/'
    
    headers = {
        'Authorization': f'Bearer {API_KEY}',
        'Content-Type': 'application/json'
    }
    
    # Create a scan
    response = requests.get(
        f'{BASE_URL}/scans',
        headers=headers,
    )
    
    if response.status_code == 201:
        scan = response.json()
        print(scan)
    ```
  </Accordion>
  <Accordion title="JavaScript/Node.js" icon="js">
    ```javascript
    const axios = require('axios');
    
    const client = axios.create({
      baseURL: 'https://api.enforster.ai/',
      headers: {
        'Authorization': `Bearer ${process.env.ENFORSTER_API_KEY}`,
        'Content-Type': 'application/json'
      }
    });
    
    // Create a scan
    async function createScan(repositoryUrl) {
      try {
        const response = await client.get('/scans');
        return response.data;
      } catch (error) {
        console.error('Error:', error.response.data);
      }
    }
    ```
  </Accordion>
  <Accordion title="cURL" icon="terminal">
    ```bash
    # Set your API key
    export ENFORSTER_API_KEY="enf_your_api_key_here"
    
    # Get scan results
    curl -X GET https://api.enforster.ai/v1/scans/{scan_id} \
      -H "Authorization: Bearer $ENFORSTER_API_KEY"
    ```
  </Accordion>
