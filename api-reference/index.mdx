## 🏃‍♂️ **Quick Start**

<Steps>
  <Step title="🔑 Get API Key">
    Generate your API key from the [Enforster AI Dashboard](https://app.enforster.ai/settings/api)
  </Step>
  <Step title="🔍 Run Your First Scan">
    ```bash
    curl -X GET https://api.enforster.ai/scans \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json"
    ```
  </Step>
  <Step title="📊 Get Results">
    Retrieve **AI-analyzed vulnerabilities** in JSON format
  </Step>
</Steps>

## 🛡️ **Security & Best Practices**

<Warning>
  **Keep your API keys secure\!** Never expose them in client-side code or public repositories.
</Warning>

<Tabs>
  <Tab title="🔐 Authentication">
    - Use **Bearer token authentication** for all requests
    - Rotate API keys regularly (90-day recommended)
    - Implement proper error handling for 401/403 responses
    - Use HTTPS only - never send keys over HTTP
  </Tab>
  <Tab title="🔄 Error Handling">
    - All errors return structured JSON responses
    - Use HTTP status codes for error classification
    - Implement exponential backoff for retries
    - Log API responses for debugging
  </Tab>
</Tabs>
